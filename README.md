🛒 Online-Shopping-Management-System
This project is a Database Management System (DBMS) designed for an online shopping platform, handling products, customers, orders, payments, and inventory efficiently.

📌 Key Features

✔ User Management – Register, login, and profile management for customers & admins.

✔ Product Catalog – Add, update, delete, and search products with categories.

✔ Shopping Cart – Users can add/remove items before checkout.

✔ Order Processing – Track orders (Pending, Shipped, Delivered).

✔ Payment System – Simulates payment integration (COD, Card, UPI).

✔ Admin Dashboard – Manage users, products, and sales reports.

✔ Database Backend – Built with MySQL/PostgreSQL for efficient data handling.




🛠 Technologies Used
Backend
Database: Microsoft SQL Server (via SQL Server Management Studio - SSMS)

Server Runtime: Node.js (Express.js or similar framework)

Frontend (if applicable)
Languages: HTML, CSS, JavaScript

Frameworks/Libraries: React.js/Angular/Vue.js (optional)
